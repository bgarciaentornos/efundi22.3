Challenge

In response to challenges from the user-base that adopts SCORM, the Advanced Distributed Learning (ADL) Initiative has identified a need to provide guidance about implementation of SCORM. Currently, little public information about implementing SCORM in a technical environment exists.

SCORM defines run-time communication requirements within the browser’s Document Object Model (DOM) using the JavaScript API. With the introduction of a third party component, this communication is slightly complicated.

ADL developed this content collection to address the practical implementation issues surrounding SCORM and development using plug-in technologies.
